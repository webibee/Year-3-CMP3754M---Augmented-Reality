# Museum AR – Educational Augmented Reality Dinosaurs
- Course: CMP3754M – Virtual and Augmented Reality
- Institution: University of Lincoln
- Grade Received: 77%
- Platform: Meta Quest 2 (Oculus)
- Engine: Unity
- Target Audience: Children (museum visitors)

# Overview
Museum AR is an educational augmented reality mobile application designed to enhance the museum experience for young visitors. The app allows children to spawn life-size, to-scale 3D dinosaur models into their physical environment using image markers and plane detection.

When a user points their phone at a museum information panel (or any flat surface), the app detects the plane and places a realistic dinosaur model in the physical space, accompanied by:
- Life-size dinosaur models (Velociraptor, Pachycephalosaurus)
- Dinosaur audio calls for immersive engagement
- Non-diegetic GUI with help instructions
- AR image marker and plane detection for accurate placement

The app is designed to bridge the gap between static museum exhibits and interactive digital learning, making history and natural science more engaging for young audiences.

# Key Features
- Plane detection -	AR Foundation / ARCore plane detection
- Image markers -	Triggers dinosaur spawning from museum panels
- Real-time 3D placement - Objects registered in physical 3D space
- Life-scale models - Dinosaurs appear at realistic sizes
